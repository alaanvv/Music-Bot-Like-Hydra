# Usage

1. [Invite it](https://discord.com/oauth2/authorize?client_id=1075021539180814407&permissions=8&scope=bot+applications.commands)
2. Create a new channel for it and use the command `/setup`  

# Instalation (for devs)

1. Download tihs repository
2. Run `npm -i`
